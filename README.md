🛠️ Technologies   

**Compose** - for awesome ui and better performance than xml.                                               
**Coroutines** - for asyncronous programming.                                                
**Retrofit** - for rest api.                                                                                                                           
**MVI** - for clear codeing and pattern.                                                                                               
**Multi moduling** - for clean Architecture.
